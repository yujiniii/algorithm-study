# 구현(implementation)

머릿속에 있는 알고리즘을 소스코드로 바꾸는 과정  
풀이를 떠올리는 것은 쉽지만, 소스코드로 옮기기 어려운 문제

### 유형

- **완전탐색**  
  모든 경우의 수를 주저없이 다 계산하는 방법
- **시뮬레이션**  
  문제에서 제시한 알고리즘을 한 단계씩 차례대로 직접 수행하는 방법

### python 에서 리스트 용량 제한

| 데이터 개수 | 메모리 사용량 |
| ----------: | ------------: |
|       1,000 |        약 4kb |
|   1,000,000 |        약 4mb |
|  10,000,000 |       약 40mb |
